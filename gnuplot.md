# gnuplot Cheatsheet

## Shorthand / Command Aliases
- linecolor: lc
- plot: p
- pointtype: pt
- replot: rep
- title: t
- with lines: w l
- with points: w p

## Lines
The following commands must be preceded with a `with lines`
```gnuplot
linecolor rgb "black" # set line color
dashtype 2 # set lines to dashtype pattern 2
```

## Output / Plot Size
```gnuplot
set size square # force plot to square
```
