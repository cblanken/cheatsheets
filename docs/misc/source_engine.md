## General / Server
`sv_cheats [0|1]`: sets `sv_cheats` on/off. `sv_cheats` is necessary to before some actions

## Players
`kick <user>`: kicks user
`kick_all`: kicks all players except local server host

## Weapons
`give <weapon>`: spawns weapon on top of the player
